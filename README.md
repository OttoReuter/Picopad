# Picopad

Es geht hier um Projekte mit dem Picopad von https://github.com/Pajenicko/Picopad. Ich habe den Bausatz zunächst mit einem RP2350-Plus von Waveshare anstelle des mitgelieferten
RP2040W bestückt. Wenn man nicht die vom Entwickler bereitgestellte SDK verwendet, kann man aus dem Datenblatt die Pinbelegung entnehmen, welche von anderen Herstellern leicht abweicht.
